# Тестовое задания для компании "Квартирка"

Технологии, использованные в проекте: 
- Next (app директория, клиентские и сервервные компоненты, router)
- Typescript
- Scss modules
- Yarn

Для запуска проекта:

Если не установлен yarn:
```
npm install -g yarn
```
Далее:
```
yarn
```
```
yarn dev
```
